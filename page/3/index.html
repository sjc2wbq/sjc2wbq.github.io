KVO

key Value Observer

为想要观察的对象添加观察者

  • object:被观察对象
  • observer:观察对象
  • forKeyPath:需要检测的属性(property的name),如UIViewV的frame,center
  • context:任意类型(一种标识,可用于同时监听不同对象的相同属性？)

  • options:

添加观察键值变化的处理方法

  • KeyPath:对应forKeyPath
  • object: 被观察者
  • change: 对应相应选项下值的改变(字典类型)可通过对应的key取值,key有new,old,等,可打印输出观察取key值
  • context: 对应context
